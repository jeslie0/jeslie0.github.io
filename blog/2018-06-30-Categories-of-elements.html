Categories of elements

James Leslie - Categories of elements
Posted on June 30, 2018 by James Leslie

When learning about colimits and presheaf categories, one often meets a category of elements. The definition of the category isn’t particularly hard to understand or remember; however, I didn’t immediately have an intuition for why it is useful or where the definition came from. This has lead me to play with it and try to understand some properties of it.

Given a locally small category 𝔸 \mathbb{A} and a functor X:𝔸op→𝐒𝐞𝐭 X:\mathbb{A}^{op} \rightarrow \mathbf{Set}, the category of elements of X X, denoted 𝔼(X) \mathbb{E}(X) or ∫𝔸X \int^\mathbb{A} X, is defined as follows:

  • Objects are pairs (A∈𝔸,x∈X(A)) (A \in \mathbb{A}, x \in X(A)),
  • Morphisms f:(A,x)→(A′,x′) f:(A, x) \rightarrow (A',x') are maps f:A→A′∈𝔸 f:A \rightarrow A' \in \mathbb{A} such that (Xf)(x′)=x (Xf)(x')=x.

Every category of elements has a projection functor P:𝔼(x)→𝔸 P:\mathbb{E}(x) \rightarrow \mathbb{A} associated with it that sends (A,x)↦A (A,x) \mapsto A and f↦f f \mapsto f. There is a useful property tucked away in the definition: for any 𝔸 \mathbb{A}-arrow f:A′→A f:A' \rightarrow A, there is a unique element x′∈X(A′) x' \in X(A') such that there is an 𝔼(X) \mathbb{E}(X)-morphism f:(A′,x′)→(A,x) f:(A',x') \rightarrow (A, x), namely x′=(Xf)(x) x' = (Xf)(x). As a result of this, we will often write morphism as f:(A′,(Xf)(x))→(A,x) f:(A', (Xf)(x)) \rightarrow (A,x). Another observation is that we can write 𝔼(X) \mathbb{E}(X) as a comma category.

There is an isomorphism 𝔼(X)≅(1⇒X) \mathbb{E}(X) \cong (1 \Rightarrow X).

We can use the category of elements to relate representablity to the existence of a terminal object.

[1, Exercise 6.2.23] Let X X be a presheaf on a locally small category. X X is representable if and only if 𝔼(X) \mathbb{E}(X) has a terminal object.

The category 𝔼(X) \mathbb{E}(X) has a terminal object if and only if there is an object (A,x) (A, x) such that for any (A′,x′) (A',x'), there is exactly one morphism f:(A′,x′)→(A,x) f:(A',x') \rightarrow (A,x). This is equivalent to there being an A∈𝔸 A \in \mathbb{A} and x∈X(A) x \in X(A) such that for all A′∈𝔸 A' \in \mathbb{A}, x∈X(A′) x \in X(A'), there is a unique morphism f:A′→A f:A' \rightarrow A such that (Xf)(x)=x′ (Xf)(x) = x'. This condition is equivalent to X X being representable, by [Corollary 4.3.2, 1].

[1, Theorem 6.2.17] Let 𝔸 \mathbb{A} be small and X:𝔸op→𝐒𝐞𝐭 X:\mathbb{A}^{op} \rightarrow \mathbf{Set} a presheaf. Then X X is the colimit of the following diagram:

We should first note that this does make sense; as 𝔸 \mathbb{A} is small, so is 𝔼(X) \mathbb{E}(X), hence a colimit does indeed exist.

This is an application of the dual of [Equation 6.2, 1].

The property that morphisms in 𝔼(X) \mathbb{E}(X) have is vital to this proof and, to myself atleast, shows why we need to use this particular small category.

Given a set S S, there is an equivalence of categories 𝐒𝐞𝐭/S≃𝐒𝐞𝐭S \mathbf{Set}/S \simeq \mathbf{Set}^S, where the latter has as objects S S indexed tuples of sets. Given (A,f:A→S)∈𝐒𝐞𝐭/S (A, f:A \rightarrow S) \in \mathbf{Set}/S, we form the tuple (f−1(s))s∈S (f^{-1}(s))_{s \in S} and given a tuple (As)s∈S (A_s)_{s \in S}, we form the disjoint union ∐s∈SAs \coprod_{s \in S}A_s along with the function g:∐s∈SAs→S g:\coprod_{s \in S}A_s \rightarrow S that sends every element in each As A_s to s s. This equivalence can be abstracted to categories by the following theorem.

[2, Proposition 1.1.7] Let 𝔸 \mathbb{A} be a small category and X:𝔸op→𝐒𝐞𝐭 X: \mathbb{A}^{op} \rightarrow \mathbf{Set} a presheaf on 𝔸 \mathbb{A}. Then there is an equivalence of categories:

[𝔸op,𝐒𝐞𝐭]/X≃[𝔼(X)op,𝐒𝐞𝐭] [\mathbb{A}^{op}, \mathbf{Set}]/X \simeq [\mathbb{E}(X)^{op}, \mathbf{Set}].

In conclusion, the category of elements is vital to proving that any presheaf is actually a colimit of a certain diagram. It also shows that slice categories of presheaf categories can be seen as a presheaf category themselves.
